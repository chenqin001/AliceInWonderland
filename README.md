# AliceInWonderland
A web based digital artwork written in TypeScript 

Feel free to use it to decorate your own website :)

Hosted using Firebase:
https://aliceinwonderland-609ed.firebaseapp.com/

To run this project:
1. install Node.js(npm) and TypeScript(version 3.5 or later) from https://nodejs.org/en/ and http://www.typescriptlang.org/
2. ```sudo npm i -g --only=prod https-localhost```
3. cd into the project directory, start the server with ```serve``` command
